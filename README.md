## ToonCraft Pro — React Native Cartoon Video Maker

A professional cartoon video maker app built with React Native.

### Features
- 15+ cartoon styles (Anime, Comic, Watercolor, Cyberpunk...)
- Full timeline editor with multi-track audio
- 100+ characters & templates
- AI scene generation & voice lip-sync
- 4K export, no watermark (Pro)
- Portrait & landscape support

### Subscription
- Free: 1 video, watermark, 720p
- **Pro ₹99/month**: Unlimited, 4K, all features

### Setup
```bash
npm install
npx react-native run-android
```
